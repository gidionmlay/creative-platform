# G DESIGN

A complete creative agency management platform.

## Features

- Multi-role authentication
- Admin Dashboard
- Client Dashboard
- Student Dashboard
- Services CMS
- Request Management
- Project Workspace
- Notifications
- Learning Management System

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Django
- Django REST Framework
- Channels
- SQLite/PostgreSQL

## Installation

Backend:

cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

python manage.py migrate
python manage.py runserver

Frontend:

Serve frontend using local server.
